# Paper Bonsai

An ESP32-based weather station displaying environmental sensor data alongside an ASCII art bonsai tree generated by a `cbonsai` algorithmic port onto a 7.5-inch 3-color Waveshare E-Ink panel.

## Setup in VSCodium / Cursor (Open VSX)

Because the official PlatformIO IDE extension relies on Microsoft proprietary C/C++ extensions, it is not available by default on the Open VSX registry used by forks like VSCodium or Cursor.

### How to Install PlatformIO IDE

1. Download the latest `.vsix` file from the [PlatformIO IDE GitHub Releases page](https://github.com/platformio/platformio-vscode-ide/releases).
2. Inside your IDE, go to the **Extensions** sidebar.
3. Click the `...` menu at the top right of the extensions pane.
4. Select **Install from VSIX...** and select the file you just downloaded.

### Command Line Alternative
Alternatively, you can just use the PlatformIO Core CLI:
1. Open a terminal and run `pip install platformio`
2. Run `pio run` to compile.
3. Run `pio run -t upload` to flash to your ESP32.
